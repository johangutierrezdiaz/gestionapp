REGISTRO CANTADAS GESTORES BEPS

El registro de las produccion diaria de los gestores BEPS se hace a traves del formato que se lee directamente desde el correo electronico. El procedimiento es el siguiente:

  • Cargue Base Día: De acuerdo a una fecha seleccionada, se carga la informacion contenida en la Base BEPS Cantadas
  • Edición Base Día: Esta Base se puede editar directamente sobre la APP o se puede enviar via mail al correo seleccionado para su posterior diligenciamento
  • Almacenamiento Cambios Base Día: Los cambios realizados sobre la base dia se deben guardar con el boton "Guardar datos >>"
